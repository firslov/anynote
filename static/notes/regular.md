# 正则表达式

- . 匹配任意一个字符，除了断行
- ... 连续三个字符，从最后一个已匹配的字符开始
- \. 转义字符，普通的.
- \w 匹配数字、大小写字母和下划线\_
- \W 与\w 相反
- \d 只匹配数字
- \d\.\d 匹配小数
- \D 与\d 相反
- \s 匹配空格、制表符、断行（空白字符）
- [~] 匹配范围、集合
- [a-z][a-z][0-9]
- [\u4e00-\u9fa5] 匹配中文字符，Unicode
- \+ 重复一次或多次
- \* 可以出现 0 次或多次
- ? 出现 0 次或 1 次
- {2,4} 重复 2 到 4 次
- \b 一个单词的边界（空格、短行、tab）
- {3} 正好重复 3 次
- {3,} 至少出现 3 次
- \+ \* ? 全是{}的快捷方式
- ( ) 分组，$1代表第1个分组，$2 代表第 2 个分组
